This project can be used to display and resize images into our frontend with the size set via url parameters.


 


   Start the server:
   npm run dev

   formatting
   npm run prettier

   Linting   
   npm run lint

   Testing
   npm run test


 Homepage endpoint   :
 http://localhost:3000

 Url
 http://localhost:3000/api/image?imageName=fjord&width=300&hight=400
